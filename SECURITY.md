# Security Policy

Aounak is a hackathon MVP. Please do not publish secrets, private keys, real emergency data, real phone numbers, or medical details in issues, pull requests, screenshots, or commits.

For project-specific security notes, see [docs/SECURITY.md](docs/SECURITY.md).

## Supported Version

This repository currently supports the Tatweer Hackathon 2026 submission branch only.

## Reporting

Open a GitHub issue with minimal public detail, or contact the maintainers privately if private contact information has been shared with you. Do not include credentials or sensitive personal data in public reports.
