# Demo Script

## 0:00-0:20 - Problem

Al Qua'a is spread across desert and farm areas. In an urgent situation, the first challenge is not only calling for help, but sharing the exact location and reaching the right nearby helper.

## 0:20-0:45 - Solution

Aounak is a mobile-first emergency response PWA. It lets a resident send a visual SOS with location and details, then lets nearby responders see the request and act.

## 0:45-1:20 - Requester Flow

Show the welcome screen, SOS selection, emergency report page, GPS/fallback coordinates, live SOS action, and optional request detail blocks.

## 1:20-1:50 - Responder Flow

Show profile readiness, responder skills, active incident visibility, required skills, accept action, and history/activity tracking where authenticated demo access is available.

## 1:50-2:15 - Offline/Fallback

Show the offline SMS fallback. Emphasize that it prepares an SMS with emergency type, contact/details, and a map link. Do not send a real SMS in the demo.

## 2:15-2:40 - Why It Scores

Aounak is relevant to Challenge 2, feasible with standard web infrastructure, and scalable to other rural communities by changing emergency categories, responder skills, and local deployment settings.

## Judge-Safe Wording

- Say: "The MVP demonstrates emergency creation, responder visibility, and fallback messaging."
- Say: "The route view is a hackathon prototype."
- Avoid: "Aounak guarantees faster emergency response."
- Avoid: "This replaces official emergency services."
